스타일 시트

        HTML 문서에 다양한 레이아웃과 꾸미기는 위한 표현적인 언어 CSS 사용
        서식 : 선택자{속성:값; ....}
        선택자 종류는
            1) 요소명 : body tag 내의 특정 tag를 직접 지적해서 선택
            2) .      : class 선택자 - 클래스 속성값으로 보유하고 있는 요소를 선택
            3) #      : id 선택자로 단수 선택 가능
        * 스타일은 하위 요소에 상속된다

        CSS 적용 방법 : inline, internal - 현재 사용하고있는 html파일에만 적용
                        external    - 여러개의 HTML파일에 동일한 css를 적용하고싶을때
                                    별도의 폴더로 파일을 적용하는게 좋다.
